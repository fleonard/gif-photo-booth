# GIF PHOTO BOOTH

## Setup

### Install via the project folder

```sh
$ npm i
```

## Development

To run develop

```sh
$ npm run start:express
```

## Tools used

- React (view)
- Redux (state)
- Redux Sagas (side effects)
- Ramda (setting state, immutability, utilities)
- Webpack (bundling)
- Babel.JS (ES6 transpiling)
- PostCSS (CSS processing)
- CSSNext (Tomorrow's CSS today!)
